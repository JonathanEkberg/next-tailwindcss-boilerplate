# Next.js with Tailwind CSS, Next-PWA and Zustand

Template for Next.js with Tailwind CSS, Next-PWA and Zustand.

## Requirements

- Node.js
- npm or yarn

## Running locally

### Installing dependencies

```
// with npm
npm install

// with yarn
yarn install
```

### Running development environment

```
// with npm
npm run dev

// with yarn
yarn dev
```

### Running production environment

```
// with npm
npm run prod

// with yarn
yarn prod
```
